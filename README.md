# TicTacToe  
    
  Welcome to Tic-Tac-Toe!  
  This game uses a zero-index.   
  Board consists of 3 columns and 3 rows:  
      (0, 0)(0, 1)(0, 2),  
      (1, 0)(1, 1)(1, 2),  
      (2, 0)(2, 1)(2, 2)  
    
  Player X: go run server.go  
  Player O: go run client.go  
    
  Enjoy playing!
